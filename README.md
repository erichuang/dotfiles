# Dot Files


## Configuration

Edit the [config.yml](https://github.com/erichuang/dotfiles/blob/master/config.yml) file.


## Building (without installation)

```terminal
rake build
```

The generated files are placed in the build/ directory.


## Installation (including building)

```terminal
rake install
```
